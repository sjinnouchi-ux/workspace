# 20260528-001: api-monitor Phase 1〜2 セットアップ＆DB初期化

## 目的

`api-monitor/` プロジェクトの Phase 1〜2 が完成したので、CLI側で以下を実施する：

1. 依存パッケージのインストール（`requirements.txt`）
2. `.env.example` → `.env` のコピー（APIキーは陣内さんが後で記入）

## 実行内容

このスクリプト `20260528-001.py` は以下を順に実行します：

- `pip install -r api-monitor/requirements.txt --break-system-packages`
- `.env` が無ければ `.env.example` をコピーして作成（APIキーは後で記入）
- `python3 api-monitor/db.py` 相当のDB初期化（`api_call_logs` / `api_keys` / `api_settings` テーブル作成）
- `app.py` / `monitor.py` / `settings.py` / `api_clients/*.py` の `py_compile` チェック
