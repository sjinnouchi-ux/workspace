# Search Console MCP 設定

## 背景・目的
CoworkとCLIの両方でGoogle Search Console MCPを使えるようにする。
`google-searchconsole-mcp` はOAuth内蔵のため、Google Cloudプロジェクト作成不要。

## 実行環境
- Python: 3.13.3
- 依存パッケージ: なし（標準ライブラリのみ）

## Step 1：~/.claude.json の現在の内容確認

```bash
cat ~/.claude.json
```

結果をCoworkに貼り付けること。

## Step 2：CLI（~/.claude.json）にMCP設定を追記

```python
import json
import os

path = os.path.expanduser("~/.claude.json")

with open(path, "r") as f:
    config = json.load(f)

if "mcpServers" not in config:
    config["mcpServers"] = {}

config["mcpServers"]["gsc"] = {
    "command": "npx",
    "args": ["-y", "google-searchconsole-mcp"]
}

with open(path, "w") as f:
    json.dump(config, f, indent=2, ensure_ascii=False)

print("✅ ~/.claude.json にSearch Console MCP追記完了")
```

## Step 4：設定確認


## 備考
- 初回起動時にブラウザが開いてGoogleアカウントのOAuth認証が必要
- 認証完了後はCowork・CLI両方からSearch Consoleデータを取得可能
- Claude Desktopの再起動が必要（Cowork側）
