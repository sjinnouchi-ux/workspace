# Handoffs

Shogunが作成した設計Markdownを保存する場所です。

Codexはここに置かれたMDを読み、対象プロジェクトの現状確認、実装、検証、報告を行います。

## 保存名

```text
YYYY-MM-DD-<topic>.md
```

例:

```text
2026-06-25-api-monitor-monthly-review.md
```

## 必須内容

- 目的
- 対象プロジェクトまたは対象リポジトリ
- 実行してよい範囲
- 実行してはいけない範囲
- 変更候補ファイル
- 検証条件
- 完了時に更新するログ
- ユーザー確認が必要な項目
