### 2. 売買ログシート（`売買ログ`）
> 全取引を記録するメインデータベース。保有判定の基準シート。

| 列 | 項目名 | 型 | 説明 |
|---|---|---|---|
| A | No | 数値 | 連番 |
| B | 買付日 | 日付 | 購入した日付 |
| C | 銘柄 | 文字列 | ティッカーシンボル（例：QQQ） |
| D | 口座種別 | 文字列 | NISA / 特定口座 |
| E | 株数 | 数値 | 購入株数 |
| F | 買付単価(USD) | 数値 | 購入時の1株価格（ドル） |
| G | 為替レート | 数値 | 購入時のUSD/JPYレート |
| H | 買付金額(円) | 数値 | 購入金額（円換算） |
| I | 売却日 | 日付 | 売却した日付（空欄＝保有中） |
| J | 売却単価(USD) | 数値 | 売却時の1株価格（ドル） |
| K | 売却金額(円) | 数値 | 売却金額（円換算） |
| L | 損益(円) | 数値 | 確定損益（円） |
| M | 損益率(%) | 数値 | 損益率 |
| N | NISA枠使用額(円) | 数値 | NISA口座の場合の使用額 |
| O | 備考 | 文字列 | 自由記載 |

**保有中の判定ロジック（Pythonより）：**
```python

sell_date = row[8]  # I列（インデックス8）
if buy_date and not sell_date:
    # 保有中として処理
```
