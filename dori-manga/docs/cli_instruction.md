# どり看護師 漫画パイプライン — CLI実行手順書

新しいPDFを追加して漫画コマ案をスプレッドシートに書き込む際の手順です。

---

## 事前確認

- `dori-manga/` フォルダに `credentials.json` と `token.json` が存在すること
- `.env` に `GOOGLE_SHEETS_ID` が設定済みであること
- 対象PDFが Google Drive の所定フォルダに追加済みであること

---

## STEP 1 ── CoworkでPDFを読み込み、コマ案JSONを生成

Cowork（このアシスタント）に以下の内容を伝えてください：

---

```
以下のPDFを読み込んで、どり看護師の4コマ漫画イントロ案を生成してください。

PDFのURL: https://drive.google.com/file/d/<FILE_ID>/view
PDFのドライブID: <FILE_ID>

出力フォーマット（JSON）:
{
  "title": "タイトル（10文字以内）",
  "pdf_id": "<GoogleドライブのファイルID>",
  "pdf_url": "https://drive.google.com/file/d/<FILE_ID>/view",
  "summary": "内容の要約（2〜3文）",
  "panel1": "1コマ目の案（場所・キャラ・状況）",
  "panel2": "2コマ目の案（問題・アラーム発生）",
  "panel3": "3コマ目の案（疑問・困惑）",
  "panel4": "4コマ目の案（どりが解説へ誘導）",
  "post_text": "Instagram投稿文章案（絵文字可・200文字以内）"
}

生成したJSONを output.json に追記してください（配列形式）。
```

---

## STEP 2 ── ターミナルでスプレッドシートに書き込む

Coworkがoutput.jsonを更新したら、ターミナルで実行：

```bash
cd ~/path/to/dori-manga
python src/write_to_sheets.py output.json
```

実行結果の例：

---

## STEP 3 ── 画像格納フォルダをR列に記入

スプレッドシートのR列（画像格納フォルダ）に、制作素材を保存するGoogleドライブフォルダのURLを貼り付けてください。

Coworkに依頼する場合は以下を伝えてください：

```
Googleドライブの「どり看護師_画像素材」フォルダ（URL: <FOLDER_URL>）の中に、
「<タイトル名>」というフォルダを作成して、そのURLをスプレッドシートのR列に記入してください。
```

---

## スプレッドシートの列構成（参考）

| 列 | 内容 |
|----|------|
| A | 完了（ドロップダウン） |
| B | タイトル |
| C | PDF URL |
| D | 内容要約 |
| E | 1コマ目_案① |
| F | 1コマ目_修正依頼 |
| G | 1コマ目_最終案 |
| H | 2コマ目_案① |
| I | 2コマ目_修正依頼 |
| J | 2コマ目_最終案 |
| K | 3コマ目_案① |
| L | 3コマ目_修正依頼 |
| M | 3コマ目_最終案 |
| N | 4コマ目_案① |
| O | 4コマ目_修正依頼 |
| P | 4コマ目_最終案 |
| Q | Instagram投稿文章案 |
| R | 画像格納フォルダ |

---

## 初回セットアップ（新しいスプレッドシートが必要な場合のみ）

```bash
cd ~/path/to/dori-manga
python src/setup_sheets.py
```

出力されたスプレッドシートIDを `.env` の `GOOGLE_SHEETS_ID` に設定してください。
