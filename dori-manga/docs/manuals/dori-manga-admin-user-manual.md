# どり漫画DB 管理画面 操作マニュアル

対象URL: https://dori-manga-admin.pages.dev/

## 1. ログイン

1. ブックマークから管理画面を開きます。
2. パスワードを入力して「ログイン」を押します。
3. 画面上部に「制作」「インポート」「プロンプト」「管理」の4タブが表示されればログイン完了です。

## 2. 制作タブ

制作タブでは、漫画の一覧と各コマ案を管理します。

- 新しい漫画を作る: タイトルを入力して「作成」を押すと、DBに作品と4コマ枠が作られ、Driveに作品フォルダも作られます。
- 状態: `完成` / `未完成` / `不採用` を選ぶと自動保存されます。
- 作業チェック: 今作業している作品を1件だけ選びます。インポートタブの作品選択に反映されます。
- 1〜4コマ案、投稿案、PDF URL: 入力すると自動保存されます。
- Driveフォルダ: 作品フォルダを開くリンクです。

## 3. CSVインポート

過去作品やまとめて作る作品はCSVで登録できます。

1. 制作タブで「CSVインポート」を押します。
2. CSVファイルを選びます。
3. 「取込を実行」を押します。
4. 結果に `新規作成 n件` / `更新 n件` / `スキップ n件` が表示されます。

CSVの列は次の順番です。

`タイトル,状態,1コマ案,2コマ案,3コマ案,4コマ案,投稿案,PDF URL`

状態は次の4つです。

- `新規作成`: 新しい作品として登録し、Driveフォルダも作ります。
- `完成`
- `未完成`
- `不採用`

同じタイトルがすでにある場合、`新規作成` はスキップされます。

## 4. インポートタブ

インポートタブでは、ChatGPTで作った画像と評価JSONを登録します。

1. 作品を選びます。
2. コマ番号を選びます。
3. 判定 `OK` / `NG` / `CLOSE` を選びます。
4. 画像ファイルを選びます。
5. ChatGPT JSONを貼り付けます。
6. 「画像をDriveへアップロードしてDB登録」を押します。

JSONが正しくない場合でも、画像はDriveに格納され、DBには「JSON読取NG」として記録されます。あとでCodexまたはCoworkで補修します。

## 5. 評価プロンプトのコピー

インポートタブ上部の「OKをコピー」「NGをコピー」「CLOSEをコピー」を押すと、最新の評価プロンプトがコピーされます。

## 6. プロンプトタブ

プロンプトタブでは、参考画像とプロンプトのセットを登録し、ChatGPTへの指示に使えます。

1. タイトルを入力します。
2. 参考画像を選びます。画像なしでも登録できます。
3. プロンプトを入力します。
4. 「登録する」を押します。

画像は登録時に自動縮小されるため、通常は元画像サイズを気にしなくて大丈夫です。

登録済みプロンプトはカード形式で表示されます。

- 画像: クリックすると拡大し、もう一度クリックすると戻ります。
- コピー: プロンプトをクリップボードにコピーします。ChatGPTへ貼り付けて使います。
- 編集: タイトルとプロンプトは直接編集できます。入力後に自動保存されます。
- 検索: タイトルの一部で絞り込めます。
- 削除: 確認後、登録データとStorage画像を削除します。

## 7. 管理タブ

管理タブでは次を確認できます。

- Supabase疎通
- 最終DB登録日時
- 作品数、完成数、未完成数、不採用数、画像評価数、改善ルール数、補修対象件数
- Drive親フォルダ
- 評価プロンプト編集
- CSVテンプレートダウンロード

## 8. 困ったとき

- ログインできない: パスワードを確認してください。
- パスワードを忘れた: 聡さんがリセットできます。
- Supabase疎通NGが出る: 管理タブが赤表示になります。聡さんに連絡してください。
- Driveフォルダがない: 画面に出る「フォルダ作成を実行」を押してください。
- Driveへの振り分け: 自動です。手動でフォルダ移動する必要はありません。
- JSONエラーが出る: そのまま登録して大丈夫です。DBに補修対象として残ります。
- 画像サイズが大きい: インポートタブの画像は10MBまでです。プロンプトタブの画像は自動縮小されます。
- プロンプトタブで画像を登録できない: PNGかJPEGで保存し直してください。
- 画面が古い: ブラウザを再読み込みしてください。
